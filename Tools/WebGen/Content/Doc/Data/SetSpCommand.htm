Typical scenario of using DbManager includes the following steps:

  • Create a DbManager class instance.
  • Set a SQL command or a stored procedure name.
  • Call an ExecuteXXX method.
The SetSpCommand method is used to set a stored procedure name and provide command parameters, if any.
